Abstract

Xypher Bot is a fully autonomous robot designed to automate height and distance measurement with precision. By integrating affordable components like an MPU6050 IMU, ultrasonic sensors, and a laser pointer, the bot utilizes fundamental trigonometry to achieve impressive accuracy. The upper module for measurement is fully tested, while the lower mobility module is in the design phase. This paper details its design, methodology, and wide-ranging applications—from environmental monitoring to industrial automation—and outlines a roadmap for future enhancements including full mobility, computer vision-based laser alignment, and IoT integration via Telegram.

Proposed Computer Vision System

The lower module, currently in design, will use a sophisticated computer vision pipeline to automatically align the laser to the top of a target object.

1. Image Acquisition & Pre-processing

A camera captures a live feed. Each frame is converted to grayscale and a Gaussian Blur is applied to reduce noise.

2. Edge & Tip Detection

The Canny Edge Detection algorithm identifies object boundaries. The system then finds the topmost point of the contour to set as the target.

3. Laser Dot Detection

The image is converted to the HSV color space, and color thresholding is used to isolate the red laser dot and find its center coordinates.

4. Error Calculation

The system calculates the pixel distance (error) between the laser dot's current position and the target object tip.

5. Servo Adjustment

The calculated error is converted into an angular displacement, and the servo motor is adjusted to move the laser towards the target.

6. Real-time Feedback Loop

This process repeats in a continuous loop until the error is minimized, ensuring the laser is precisely aligned with the object's tip.
